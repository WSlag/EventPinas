# 🎉 EventPinas – Event Discovery & Booking Platform

## 🚀 In Development Platform

An event marketplace platform designed to connect event organizers and attendees in a centralized digital system.

👉 Live Preview: https://eventpinas.web.app

---

## 🧠 Business Context

Event management in the Philippines often relies on:

- Social media posts (Facebook events)
- Manual coordination via chat
- Limited centralized discovery

👉 EventPinas aims to solve this by providing a **structured event discovery and booking platform**.

---

## 💡 Key Features

### 👥 For Attendees

- Browse and discover events  
- View event details and schedules  
- Register or book events  
- Save and track interested events  

### 🎤 For Event Organizers

- Create and manage events  
- Publish event listings  
- Track attendees  
- Manage event details and updates  

---

## ⚙️ Tech Stack

- React + Vite  
- Firebase (Auth, Firestore, Storage)  
- Tailwind CSS  
- Progressive Web App (PWA)  

---

## 🏗 System Capabilities

This platform demonstrates:

- Marketplace system design (organizers ↔ attendees)
